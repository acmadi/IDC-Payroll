RTL
* Mirrors Blueprint, so it can be used with Right-to-Left languages.

By Ran Yaniv Hartstein, ranh.co.il

Usage
----------------------------------------------------------------

1) Add this line to your HTML:
   <link rel="stylesheet" href="../../../../../../static/zzzjoshuaclayton-blueprint-css-v1.0.1-8-g9bf9513/joshuaclayton-blueprint-css-9bf9513/blueprint/plugins/rtl/css/blueprint/plugins/rtl/screen.css" type="text/css" media="screen, projection">
